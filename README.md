# Blog customizer

Интерактивный редактор стилей для статей с возможностью настройки параметров отображения в реальном времени.

[Макет проекта](https://www.figma.com/file/FEeiiGLOsE7ktXbPpBxYoD/Custom-dropdown?type=design&node-id=0%3A1&mode=design&t=eXRJnWC6Xsuw0qR4-1)

## Описание проекта

Проект представляет собой веб-приложение для динамического редактирования стилей статьи. Пользователь может через боковую панель настроить визуальные параметры (шрифты, цвета, отступы), которые мгновенно применяются к контенту через CSS-переменные. Основная цель — создание удобного инструмента для визуального оформления текстового контента.

**Технологический стек:**

- **Frontend:** React, TypeScript
- **Стилизация:** CSS Modules, CSS Custom Properties
- **Инструменты разработки:** Storybook (для документирования компонентов), ESLint, Stylelint, Prettier
- **Сборка:** Webpack

## Topics

`react` `typescript` `css-variables` `storybook` `form-handling` `state-management` `component-library`

## Выводы и результаты

В процессе реализации проекта были освоены ключевые концепции:

- **Компонентный подход:** Создание переиспользуемых UI-компонентов с использованием Storybook для документирования
- **Управление состоянием:** Организация двусторонней синхронизации между формой настроек и отображаемой статьей
- **CSS-архитектура:** Использование CSS-переменных для динамического изменения стилей в рантайме
- **Композиция компонентов:** Построение сложных интерфейсов из простых, изолированных компонентов

Основной вывод: CSS-переменные являются мощным инструментом для создания динамических интерфейсов, а четкое разделение состояния формы и состояния отображения упрощает поддержку и расширение функциональности.

## Установка и запуск

### Системные требования

- Node.js 16+
- npm или yarn

### Развёртывание

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run start

# Сборка для production
npm run build

# Запуск Storybook для просмотра компонентов
npm run storybook

# Проверка кода
npm run lint        # ESLint для TypeScript
npm run stylelint   # Stylelint для CSS
npm run format      # Форматирование кода
```

Приложение будет доступно по адресу `http://localhost:8080`
